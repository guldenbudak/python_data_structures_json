# Python Stajer Ödev Paketi

Bu klasörde 5 adet ödev ve her ödevin veri dosyaları vardır.

## Klasör Yapısı
- `odev*_*.md`: Ödev yönergeleri
- `inputs/`: Her ödevin girdi JSON dosyaları
- `outputs/`: Stajerin üreteceği çıktı dosyaları (isteğe bağlı)

## Çalışma Notu
Stajer her ödev için:
1) İlgili `inputs/*.json` dosyasını okuyacak
2) İstenen hesaplamaları yapacak
3) Konsola rapor basacak (bazı ödevlerde dosyaya çıktı kaydı bonus)

İyi çalışmalar!
